# 📊 Mutual Fund Analytics Dashboard

A complete end-to-end Data Analytics project that analyzes the Indian Mutual Fund industry using Python, SQLite, SQL, and Power BI.

The project focuses on data ingestion, cleaning, exploratory data analysis (EDA), KPI development, dashboard creation, and business insights generation to help understand fund performance, investor behavior, and industry trends.

---

# 🚀 Project Objective

The objective of this project is to build an interactive Business Intelligence dashboard that enables users to:

- Analyze mutual fund industry growth
- Evaluate fund performance
- Understand investor behavior
- Track SIP growth
- Compare Asset Management Companies (AMCs)
- Generate actionable business insights

# 💡 Business Insights

Some major insights generated from the dashboard include:

- Industry AUM has shown consistent growth.
- SIP investments continue to increase every year.
- Equity funds attracted the highest investor interest.
- Top-performing funds delivered superior risk-adjusted returns.
- Metro cities contributed the majority of investments.
- Working-age investors represented the largest investor segment.

# ▶️ How to Run the Project

Clone the repository:

git clone https://github.com/yourusername/Mutual-Fund-Analytics-Dashboard.git

Install dependencies:

pip install -r requirements.txt

Run the notebooks in order:

01_data_ingestion.ipynb

02_data_cleaning.ipynb

03_exploratory_data_analysis.ipynb

04_kpi_and_data_modeling.ipynb

Open the Power BI dashboard:

dashboard/Mutual_Fund_Analytics.pbix
